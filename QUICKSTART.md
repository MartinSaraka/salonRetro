# 🚀 Quick Start - Vintage Barbershop

## Rýchle spustenie

```bash
# 1. Nainštalujte závislosti
npm install

# 2. Spustite vývojový server
npm run dev

# 3. Otvorte v prehliadači
# http://localhost:3000
```

## Build pre produkciu

```bash
# Build
npm run build

# Preview buildu
npm run preview
```

## Čo nájdete

- **Vintage dizajn** - Starobylý vzhľad s sepia tónmi
- **Rezervačný systém** - 3-krokový booking proces
- **Responzívny** - Funguje na všetkých zariadeniach
- **SEO optimalizované** - Meta tagy a štruktúra

## Prispôsobenie

Upravte farby a fonty v:
- `tailwind.config.js` - Farebná paleta
- `assets/css/main.css` - Vintage efekty
- `nuxt.config.ts` - Fonty a meta tagy
